Abstract

The quantum computer is an extraordinary invention built by mankind as it allows humans to leverage the power of quantum physics to solve difficult mathematical problems in a short amount of time. While such advances in technology are very welcome, this hardware has the potential to crack commonly used public-key encryptions in a feasible amount of time, which would be catastrophic to most Single Sign On (SSO) systems which rely on such technologies.

The reason why most public-key encryptions have this problem is because most of our modern public-key encryptions base their difficulty on factoring large numbers. A common factoring algorithm used for this purpose is Shor's algorithm. Running Shor's algorithm on classical computer will exceed the lifetime of the universe, while a quantum computer will run Shor's algorithm in polynomial time. A new algorithm known as Regev's algorithm has proven to factor large prime numbers much faster than Shor's, which further provokes the need for a secure solution. A new branch of cryptography known as Post Quantum Cryptography (PQC) has recently emerged to address this issue.

We understand that quantum computers are still a work-in-progress, and therefore, it is important to secure modern authentication and authorization systems before quantum computers fully mature. In this paper, we will come up with a secure Single Sign On system which uses PQC for authentication and authorization. We also will propose a fast PQC key storage mechanism, a faster PQC authentication mechansim, a faster/developer-friendly signature scheme, and an improved risk-based authentication methodology to ultimately improve the security of modern authentication/authorization systems.

Results

Improved PQC Key Storage Speed Using Vector Database

The vector database-based key storage system showed significantly faster write speeds, with a 464% speed increase compared to the SQL version, especially as the entry count increased. However, the vector database used more CPU and RAM than the SQL solution.

Faster & Quantum Resistant JWT Token Generation/Verification

The JWT token generation process using Crystals-Dilithium was 90% faster than RSA (2048) and performed faster than ED25519 at verification. While it was slower at generation compared to ED25519, it outperformed in verification speed by 60%.

Enhanced Risk-Based Authentication Model With SVM

The improved risk-based authentication model, using Support Vector Machines (SVM), achieved a 98% accuracy rate in detecting authentication anomalies, surpassing the 96% accuracy of previous models like Freeman's. This increase in accuracy offers stronger protection against suspicious logins.

Faster Mutual Challenge-Response Authentication

The mutual challenge-response mechanism, built with Crystals-Kyber, completed the challenge-response and token exchange in just 142 microseconds. This represents a significant speed improvement over traditional RSA-based systems, with reduced memory usage by 58%.

The literature on cryptographic systems has increasingly focused on the vulnerabilities posed by quantum computing to traditional encryption methods, especially public-key cryptography schemes like RSA. The work of Rivest, Shamir, and Adleman laid the foundation for public-key cryptosystems, but subsequent research, including the findings of Shor, demonstrated that RSA and similar schemes are highly susceptible to quantum-based attacks, which can break their security in polynomial time. The rise of quantum computing prompted researchers to explore Post-Quantum Cryptography (PQC) as a means to mitigate these threats. Cryptographic algorithms such as Crystals-Kyber and Crystals-Dilithium, both based on the Learning With Errors (LWE) problem, have emerged as strong candidates to secure cryptosystems against quantum attacks. The literature, including work by Regev et al., has highlighted the advantages of these PQC algorithms in areas such as key generation, signature verification, and encapsulation. However, existing research has not sufficiently addressed the challenges of integrating these algorithms into Single Sign-On (SSO) systems or optimizing the key storage and retrieval processes within modern authentication frameworks.

While there is substantial research on the vulnerabilities of traditional cryptographic systems and the potential of PQC, a significant gap remains in the practical application of PQC in Single Sign-On (SSO) solutions. Most research focuses on theoretical advancements, with limited exploration of how PQC algorithms like Crystals-Kyber and Crystals-Dilithium can be deployed in real-world SSO architectures. Additionally, there has been little focus on optimizing the storage and retrieval of PQC keys using vector databases, which offer faster operations compared to SQL databases. Another gap is in improving the efficiency of challenge-response authentication mechanisms and risk-based authentication models, particularly in enhancing the accuracy and speed of detecting suspicious login attempts.

The main research problem addressed in this paper is the lack of a secure and efficient Single Sign-On (SSO) system capable of withstanding quantum-based attacks. Traditional cryptographic methods, such as RSA, are vulnerable to quantum computing, and current key storage and retrieval mechanisms are not optimized for the unique requirements of PQC algorithms. Moreover, the existing challenge-response authentication mechanisms used in SSO systems are inefficient, and risk-based authentication models often fail to detect suspicious activities with high accuracy. This research seeks to address these problems by developing a quantum-resistant SSO system that integrates PQC, optimizes key storage with vector databases, and improves authentication accuracy and speed.

The objectives of this research are to design and implement a quantum-resistant Single Sign-On (SSO) solution using Post-Quantum Cryptography (PQC) algorithms, specifically Crystals-Kyber for encryption and Crystals-Dilithium for digital signatures. The solution aims to enhance the efficiency of key storage and retrieval by using vector databases instead of traditional SQL databases. Another objective is to create a fast mutual challenge-response authentication mechanism that reduces communication overhead while maintaining high security. Finally, the research aims to improve the accuracy of risk-based authentication models by utilizing machine learning techniques such as Support Vector Machines (SVM) to detect anomalous authentication attempts with greater precision.

The research methodology begins by integrating PQC algorithms into the architecture of a federated SSO system. Crystals-Kyber is used for encryption, while Crystals-Dilithium is implemented for digital signatures. A vector database, specifically Qdrant, is employed to store and retrieve public keys, leveraging cosine similarity to ensure fast key retrieval. A benchmark comparison is conducted between the vector database and a traditional SQL database, measuring the speed of key insertions and retrievals. The research also develops a fast mutual challenge-response authentication mechanism using Crystals-Kyber, which completes authentication with fewer TCP communications. For the risk-based authentication model, the research utilizes a dataset containing 33 million authentication attempts to train machine learning models, including Logistic Regression, Random Forest, and Support Vector Machines. The goal is to improve the detection accuracy of suspicious login attempts by enhancing the model's performance.
